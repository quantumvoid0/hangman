# Hangman
- mini terminal based hangman game

## Install

```fish
git clone https://github.com/quantumvoid0/hangman
cd hangman
./setup install
clear
./hangman
```
- to play again , just run `hangman` in ur terminal

## Uninstall

```fish
git clone https://github.com/quantumvoid0/hangman
cd hangman
./setup uninstall
```
